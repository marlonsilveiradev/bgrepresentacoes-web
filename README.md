# 🚀 BG Representações Web

Frontend da aplicação BG Representações, responsável pela interface de gestão de clientes, vendas e operações comerciais.

Este projeto consome a API desenvolvida em Node.js, seguindo boas práticas de organização, componentização e integração com backend.

## 📌 Funcionalidades

- Autenticação de usuários
- Listagem de clientes
- Cadastro de clientes
- Integração com API REST
- Consumo de rotas protegidas via JWT
- Feedback visual de ações (toast/notificações)

---

## 🧠 Arquitetura

O projeto segue organização baseada em componentes, com separação de responsabilidades:

- components → componentes reutilizáveis  
- pages → páginas da aplicação  
- services → comunicação com API  
- hooks → lógica reutilizável  
- utils → funções auxiliares  

---

## ⚙️ Tecnologias utilizadas

- React
- JavaScript
- HTML5
- CSS3
- Fetch API / Axios
- React Hooks

---

## 🔐 Autenticação

A aplicação utiliza autenticação baseada em JWT:

- login via API
- armazenamento de token
- envio automático no header das requisições protegidas

---

## 🔄 Integração com Backend

O frontend foi desenvolvido para consumir uma API REST com:

- autenticação JWT
- validação de dados
- controle de acesso

## ▶️ Como executar localmente

Clone o repositório:

git clone https://github.com/marlonsilveiradev/bgrepresentacoes-web

Acesse a pasta:

cd bgrepresentacoes-web

Instale as dependências:

npm install

Execute o projeto:

npm run dev
